# simple-HTML-website-using-GitHub-Pages
Deploy a simple HTML website using GitHub Pages

Tools: GitHub Pages
Deliverables: Live website link and GitHub repo

Mini Guide:
 
1.Create an index.html file.

2.Push it to a new GitHub repo.

3.Go to repo → Settings → Pages → enable GitHub Pages.

4.Select main branch and root folder.

5.Access live website from the link GitHub gives.

6.Try customizing with CSS.
